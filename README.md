## Boilerplate Express Application

This is a boilerplate Express application using ES2015 via Babel.

Commands:
```bash
# Install and run
$ yarn
$ yarn start

# Run tests
# Tests should be run with `NODE_ENV=test` for test database configuration
$ yarn test
$ yarn test:unit
$ yarn test:http

# View code coverage
$ yarn test:coverage

# Lint code
$ yarn lint
```
